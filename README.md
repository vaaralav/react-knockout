# react-knockout

## Install

```bash
npm install --save react-knockout
```

## Usage

```jsx
import React, { Component } from 'react';

import MyComponent from 'react-modern-library-component';

class Example extends Component {
  render() {
    return <MyComponent />;
  }
}
```
